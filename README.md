# Tribu-JS Neolynk - Live coding 26/04/18

Simple demo of Cucumber-JS for functional testing with an simple express server

## Prerequisites
- [Node.JS](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/lang/en/)

## Installation
```bash
yarn install
```

## Run tests
```bash
yarn test
```
